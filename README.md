#Ricochet Robots

##Codage des éléments du plateau :

* 0 : case vide
* 1 : objectif

* 10 : mur simple gauche
* 11 : mur simple haut
* 12 : mur simple droite
* 13 : mur simple bas

* 20 : mur double gauche/haut
* 21 : mur double droite/haut
* 22 : mur double gauche/bas
* 23 : mur double droite/bas
* 24 : mur complet

##Codage des robots :

* 0 : robot rouge
* 1 : robot vert
* 2 : robot bleu
* 3 : robot gris
